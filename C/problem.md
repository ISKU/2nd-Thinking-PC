### 제목
별

### 문제
<p>출력 예제를 보고 별 찍는 규칙을 유추하여 별을 찍어 보자.</p>

### 입력
<p>첫 번째 줄에는&nbsp;정수 <em>N </em>(0 &le; <em>N</em> &le; 10)이 주어진다.</p>

### 출력
<p>별 찍는 규칙에 따라 별을 출력한다.</p>

### 예제 입력 1
```
0
```

### 예제 출력 1
```
*
```

### 예제 입력 2
```
1
```

### 예제 출력 2
```
**
*
```

### 예제 입력 3
```
2
```

### 예제 출력 3
```
****
* *
**
*
```

### 예제 입력 4
```
3
```

### 예제 출력 4
```
********
* * * *
**  **
*   *
****
* *
**
*
```

### 분류
규칙성 찾기
